PRACTICAL 10: SOAP Web Service to Convert Rupees to Dollar
AIM

To implement a SOAP Web Service that converts Indian Rupees (INR) to US Dollars (USD) using the conversion rate:

1 USD = 75 INR

The service is created in .NET and consumed using a Java Web Application.

STEP 1: Create Web Service in .NET

Where: Microsoft Visual Studio

Open Microsoft Visual Studio

Go to

File → New → Project

Select

ASP.NET Web Application

Enter

Project Name : Service

Click OK

STEP 2: Add Web Service

Where: Visual Studio

Right click project Service

Add → New Item

Select

Web Service

Name the file

Operation

Click Add

STEP 3: Write Web Service Code

Where: Visual Studio (Operation.asmx.cs)

File:

Operation.asmx.cs
Simplified Code
using System;
using System.Web.Services;

namespace Service
{

[WebService(Namespace = "http://tempuri.org/")]

public class Operation : WebService
{

[WebMethod]

public double Rsconvert(double rs)
{

// Conversion Rate
// 1 USD = 75 INR

double dollars = rs / 75;

return dollars;

}

}

}

Explanation

dollars = rupees / 75

This converts Indian Rupees to US Dollars.

Example

750 INR / 75 = 10 USD
STEP 4: Run the Web Service

Where: Visual Studio

Click

Run (Green Play Button)

Open Service Description

Copy the WSDL URL

Example

http://localhost:51594/Operation.asmx?WSDL
STEP 5: Test the Web Service

Where: Browser

Enter value

Rupees = 750

Output

Dollars = 10

Right click project
