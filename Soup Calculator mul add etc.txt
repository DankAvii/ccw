Practical 1: Calculator Web Service (SOAP) – Java
Step 1: Create Java Project

Open NetBeans / Eclipse IDE

Click File → New Project

Select Java Application

Enter Project Name:

CalculatorSOAP

Click Finish

Step 2: Create Web Service Class

Create a class:

CalculatorService.java
import javax.jws.WebService;
import javax.jws.WebMethod;

@WebService
public class CalculatorService {

    @WebMethod
    public double add(double a, double b) {
        return a + b;
    }

    @WebMethod
    public double subtract(double a, double b) {
        return a - b;
    }

    @WebMethod
    public double multiply(double a, double b) {
        return a * b;
    }

    @WebMethod
    public double divide(double a, double b) {
        if (b == 0) {
            throw new ArithmeticException("Division by zero is not allowed");
        }
        return a / b;
    }
}
Step 3: Create Server Program

Create another class:

Server.java
import javax.xml.ws.Endpoint;

public class Server {

    public static void main(String[] args) {

        Endpoint.publish("http://localhost:8080/calculator", new CalculatorService());

        System.out.println("Calculator SOAP Service Started...");
    }

}
Step 4: Run the Server

Run the file:

Server.java

Console output should show:

Calculator SOAP Service Started...
Step 5: Check WSDL File

Open browser and type:

http://localhost:8080/calculator?wsdl

If XML appears, the SOAP Web Service is running correctly.

Step 6: Create Client Application

Create a class:

Client.java
import java.net.URL;
import javax.xml.namespace.QName;
import javax.xml.ws.Service;

public class Client {

    public static void main(String[] args) throws Exception {

        URL url = new URL("http://localhost:8080/calculator?wsdl");

        QName qname = new QName("http://service/", "CalculatorService");

        Service service = Service.create(url, qname);

        CalculatorService calc = service.getPort(CalculatorService.class);

        double result1 = calc.multiply(10, 5);
        double result2 = calc.divide(20, 4);

        System.out.println("Multiplication Result: " + result1);
        System.out.println("Division Result: " + result2);

    }

}
Step 7: Run Client Program

Run:

Client.java

Output:

Multiplication Result: 50.0
Division Result: 5.0
