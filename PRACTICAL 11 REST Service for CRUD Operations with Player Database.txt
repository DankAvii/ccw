PRACTICAL 11: REST Service for CRUD Operations with Player Database
AIM

To create a RESTful Web Service that performs CRUD operations (Create, Read, Update, Delete) on a Player database with the following fields:

Player_Name
Age
Country
Runs

STEP 1: Create Web Application

Where: NetBeans IDE

Open NetBeans IDE

Click

File → New Project

Select

Java Web → Web Application

Enter

Project Name : PlayerRESTService
Server : GlassFish Server 4.1

Click

Next → Finish
STEP 2: Create RESTful Web Service

Where: NetBeans IDE

Right click project

New → RESTful Web Services from Patterns

Select Pattern

Simple Root Resource

Click Next

STEP 3: Configure Resource Class

Where: NetBeans IDE

Enter the following details

Project : PlayerRESTService
Resource Package : player
Path : player
Class Name : PlayerResource
Representation Class : java.lang.String

Click Finish

NetBeans will automatically generate the REST resource class.

STEP 4: Write REST Service Code

Where: NetBeans (PlayerResource.java)

File

PlayerResource.java
Simplified Code
package player;

import javax.ws.rs.*;
import javax.ws.rs.core.MediaType;

@Path("/player")

public class PlayerResource {

String playerName = "Virat Kohli";
int age = 35;
String country = "India";
int runs = 26000;

@GET
@Produces(MediaType.TEXT_PLAIN)

public String getPlayer(){

return "Player Name: " + playerName +
"\nAge: " + age +
"\nCountry: " + country +
"\nRuns: " + runs;

}

@POST
@Produces(MediaType.TEXT_PLAIN)

public String createPlayer(){

playerName = "Rohit Sharma";
age = 36;
country = "India";
runs = 18000;

return "Player Created";

}

@PUT
@Produces(MediaType.TEXT_PLAIN)

public String updatePlayer(){

runs = 20000;

return "Player Updated";

}

@DELETE
@Produces(MediaType.TEXT_PLAIN)

public String deletePlayer(){

playerName = "";

return "Player Deleted";

}

}
STEP 5: Deploy the Project

Where: NetBeans Server

Right click project

Deploy

GlassFish Server will start and deploy the REST service.

STEP 6: Test REST Service

Where: Browser / NetBeans REST Tester

Open browser

http://localhost:8080/PlayerRESTService/webresources/player

You can test CRUD operations using NetBeans Test RESTful Services panel or Postman.
