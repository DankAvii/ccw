PRACTICAL 4: OpenStack Installation with User and Private Network

STEP 1: Update and Upgrade the System

Open Terminal and run:

sudo apt update -y && sudo apt upgrade -y

This command updates the package list and upgrades installed packages.

STEP 2: Reboot the System

After updating the system, reboot the machine.

sudo reboot

STEP 3: Create Stack User

Create a new user named stack for installing OpenStack.

sudo adduser -s /bin/bash -d /opt/stack -m stack

Explanation
-s → sets shell
-d → home directory
-m → create home directory

STEP 4: Give Sudo Privileges to Stack User

Run the following command to give administrator privileges.

echo "stack ALL=(ALL) NOPASSWD: ALL" | sudo tee /etc/sudoers.d/stack

This allows the stack user to run sudo commands without password.

STEP 5: Switch to Stack User

Login as stack user.

su - stack

STEP 6: Install Git

Install Git if it is not already installed.

sudo apt install git -y

Git is required to download DevStack.

STEP 7: Download DevStack

Clone the DevStack repository.

git clone https://git.openstack.org/openstack-dev/devstack

This downloads the DevStack installation files.

STEP 8: Go to DevStack Directory

Move to the DevStack folder.

cd devstack

STEP 9: Create Configuration File

Create a configuration file named local.conf

vim local.conf

Press Esc + i to enter insert mode.

Add the following configuration.

[[local|localrc]]

ADMIN_PASSWORD=StrongAdminSecret123
DATABASE_PASSWORD=StrongAdminSecret123
RABBIT_PASSWORD=StrongAdminSecret123
SERVICE_PASSWORD=StrongAdminSecret123

HOST_IP=10.208.0.10

Save the file.

Press:

Esc
:wq

To find your system IP address run:

ip addr

Replace HOST_IP with your system IP.

STEP 10: Install OpenStack using DevStack

Run the installation script.

./stack.sh

If installation fails, run:

FORCE=yes ./stack.sh

This installs OpenStack services.

STEP 11: Access OpenStack Dashboard

After installation completes, open browser and enter:

http://server-ip/dashboard

Example:

http://10.208.0.10/dashboard

Login using:

Username:

admin

Password:

StrongAdminSecret123

STEP 12: Create Private Network (Optional Step)

After login to the dashboard:

Go to Project → Network → Networks

Click Create Network

Enter network name

Create Subnet

Configure Gateway and IP range

This creates a private network inside OpenStack.
